#Golang-Rest-Api-Using-Mux

## Build Setup

```bash
# step1
$  go build

# step2
$ ./Go-api-mux
 
#OR
$ go build && ./Go-api-mux

# We need gorilla + gorm + driver
# Here we are using mysql as our database

##Commend that you need to run

1.For Gorilla mux
#Run
go get -u github.com/gorilla/mux1

2.For Gorm 
#Run
go get -u gorm.io/gorm

3. For mysql driver
#Run
go get -u gorm.io/driver/mysql
